# 🌐 My Portfolio Website

This is my personal portfolio website built using React.js.
I created this project to showcase my skills, projects, and contact information in a clean and modern UI.

---

## 🚀 What I Built

This portfolio includes:

* 🏠 Hero section with my introduction
* 👩‍💻 About section describing my background
* 🛠️ Skills section with categorized technologies
* 📂 Projects section with my GitHub projects
* 📬 Contact section with a form
* 🔗 Footer with my details

---

## 🛠️ Technologies Used

* React.js
* JavaScript
* HTML
* CSS

## ⚙️ How to Run This Project

### 1. Clone the repository

```bash
git clone https://github.com/shiwani2006/Portfolio.git
```

### 2. Open the project folder

```bash
cd Portfolio
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm run dev
```

Then open the link shown in terminal (usually http://localhost:5173)
